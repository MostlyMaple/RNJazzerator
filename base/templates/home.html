Welcome to RNJazzerator!

What is RNJazzerator?

Have you ever wondered what a prime number sounds like? RNJazzerator is a program that analyzes audio and detects individual notes within the frequency range of 0 Hz to 8k Hz. The idea behind using Jazz as a primary source of audio is that jazz, by it’s nature, is non-deterministic. This means that it is often much harder to predict the audio values that are used in calculating a key, Making it overall a more secure source of audio. With that being said, any audio file can technically be used, as input, as long as the program has a suitable amount of notes to record.

How Does It Work?

Once a series of 171 musical notes are analyzed, a unique 6-bit binary value is assigned to each note. Appending all the binary values, cutting the last value in half (3 binary digits), and adding 1, we are left with a 1024-bit binary number. How can we tell if this number is a prime? We use wolfram one’s cloud API deployment to analyze the binary number to determine if it is prime or not. If it is prime, we can then move on to the next step and use it for RSA cryptography.
